Sobre Nosotros

Este proyecto consiste para que las personas puedan realizar publicaciones de videosy fotos de cosas que mas le gusten para que otras personas puedan verlo y decidir si al igual que ellos les gusta

Mision

La misión de este proyecto es poder cumplir con las expectativas del departamento de ingeniería para indicar que se puede realizar un excelente trabajo

Visión

La visión de este proyecto es darle un ejemplo a los estudiantes que busquen llevar la clase magistral de introducción a la programación y computación de como se puede realizar este tipo de poryectos.
